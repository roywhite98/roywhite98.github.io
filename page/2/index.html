条件概率
Created2023-06-04
A、B为样本空间$\varOmega$中两事件，若P(B)>0，则称：$$P(A|B)=\frac{P(AB)}{P(B)}$$为“在B发生下A的条件概率”，简称条件概率。 乘法公式：若$P(B)>0$，则：$$P(AB)=P(B)P(A|B)$$

Deploy Hexo on another computer
Created2023-06-04
在另一台电脑上部署一下Hexo。利用git分支来同步hexo。main branch放静态页面文件hexo branch放源码 设置hexo branch为default branch 在老电脑上push源文件到hexo branch 在新电脑上clone hexo branch到本地 在新电脑上安装hexo环境 在新电脑上推送源码到hexo branch，生成的静态页面仍推送到main branch，就是这么神奇 使用git add . && git commit -m “message” && git push 更新源文件 使用hexo clean && hexo g && hexo d推送静态到main 然后报了个错，看了一下主题文件夹是空的，copy一下主题过来，hexo d，完成 每次使用前pull一下以保证同步
